# OpenWorkReview — 开源职场评价库

一个基于 GitHub Issues 的开源项目，收集并存档职场个人对企业的真实评价。通过公开透明、不可篡改的 Issue 记录，打造人人可查询、可协作的企业口碑数据库。

## 🚀 项目愿景

* 构建真实可靠的企业评价体系
* 减少求职信息差，帮助人们避雷
* 让优秀企业被看见，推动职场正向循环
* 打造完全开源的职场数据基础设施

## 📌 查询与提交评价

### 🔍 查询

在 Issues 面板中搜索企业名称，即可查看他人分享的真实工作体验。

### ✏️ 提交评价

创建新的 Issue 并使用下面的模板填写你的经历。审核通过后 Issue 将被锁定，确保数据透明、不可篡改。

## 📝 Issue 提交模板

```
## 企业全名

## 工作时间段

## 职位 / 部门

## 薪资待遇
- 基本工资：
- 奖金 / 绩效：
- 五险一金 / 社保情况：
- 是否按时发放：

## 工作内容

## 工作氛围与管理
- 团队氛围：
- 上级管理风格：
- 内部沟通情况：

## 加班 & 工作强度
- 实际加班情况：
- 是否存在 996 / 经常性加班：
- 加班记录是否透明：

## 职业发展空间
- 晋升机制：
- 是否公平：

## 企业优点

## 企业缺点

## 推荐指数（1–5）

## 其他补充内容
```

## 🤝 贡献方式

* 提交企业评价（Issue）
* 优化 README / 文档
* 贡献标签体系或自动化脚本
* 协助数据统计或结构化

### 推荐标签

* positive — 企业优点
* negative — 企业缺点
* compensation — 薪资福利
* overtime — 加班情况
* culture — 企业文化
* management — 管理风格
* verified — 真实验证

## 🔒 数据与隐私

* 避免包含可识别个人身份的信息
* 不发布企业机密或违法信息
* 内容由社区贡献，维护者不对真实性承担法律责任

## 📜 免责声明

所有评价仅代表个人观点，不构成法律依据或商业事实，阅读者需自行判断内容可信度。
